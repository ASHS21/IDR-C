# Search (Cmd+K)

> For all users: quickly find identities, navigate pages, and trigger actions.

## Prerequisites

- Signed in to Identity Radar

## Opening the Command Palette

Press **Cmd+K** (macOS) or **Ctrl+K** (Windows/Linux) from any page to open the command palette.

## Search Categories

The command palette searches across three categories:

### Identities

Type a name, UPN, or email to find identities. Results show the identity name, type badge, and risk score. Click a result to navigate to the identity detail page.

### Pages

Type a page name to navigate directly. Examples:

- "tiering" navigates to AD Tiering
- "violations" navigates to Policy Violations
- "settings" navigates to Settings

### Actions

Type an action name to trigger it. Examples:

- "sync" triggers an integration sync
- "export" exports the current view

## Keyboard Navigation

| Key | Action |
|-----|--------|
| Up/Down arrows | Move selection |
| Enter | Open selected result |
| Escape | Close the palette |
| Cmd+K / Ctrl+K | Toggle open/close |

## Verification

The command palette opens instantly and returns relevant results within 200ms of typing.

## Troubleshooting

| Problem | Solution |
|---------|----------|
| Palette does not open | Ensure keyboard focus is on the Identity Radar page, not a browser URL bar |
| No identity results | Identities must be imported before they appear in search |
| Slow results | Check database connectivity |
