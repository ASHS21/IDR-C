# Keyboard Shortcuts

> All keyboard shortcuts available in Identity Radar.

## Global Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd+K` / `Ctrl+K` | Open command palette (search) |
| `Escape` | Close modal, palette, or tooltip |

## Command Palette

| Shortcut | Action |
|----------|--------|
| `Up` / `Down` arrows | Move selection |
| `Enter` | Open selected result |
| `Escape` | Close palette |

## Data Tables

| Shortcut | Action |
|----------|--------|
| `Tab` | Move to next cell or control |
| `Shift+Tab` | Move to previous cell or control |
| `Space` | Toggle row selection (when checkbox focused) |

## Navigation

Use the command palette (`Cmd+K`) to navigate to any page by typing its name:

| Type | Example |
|------|---------|
| Page name | "tiering", "violations", "settings" |
| Identity name | "Ahmed Al-Rashidi" |
| Action | "sync", "export" |
